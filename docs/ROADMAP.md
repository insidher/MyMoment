# Roadmap

## Phase 1: Foundation [COMPLETED]
*   **Core Experience**: Unified listening room for YouTube & Spotify playback.
*   **Moment Capture**: Mechanics to saving start/end times with metadata.
*   **Database**: Supabase integration for reliable storage.
*   **Auth**: Supabase Auth implementation.

## Phase 2: Refinement [IN PROGRESS]
*   **Spotify Reliability**: Improving seek logic and playback stability.
*   **Data Integrity**: Backfilling canonical IDs and ensuring unique track sources.

## Phase 3: Community
*   **Social**: Public profiles, shareable links, and "Saved By" counts.
*   **Discovery**: Feeds and recommendation engine.
